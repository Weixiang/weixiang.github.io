最近做的一些 Ghost 小优化（Ghost 博客调教）

新增评论区#

大家终于可以愉快地畅所欲言啦！

而且因为我们服务器在新加坡，不受中国大陆法律限制，也没有敏感词蛤蛤蛤

采用的是Valine作为我们的评论系统，快速、简洁且高效、无后端。 获取

在 Code Injection 的 Footer 部分填入

<script src="//cdn1.lncld.net/static/js/3.0.4/av-min.js"></script>
<script src='//unpkg.com/valine/dist/Valine.min.js'></script>
<script>
 new Valine({
  el: '#vcomments',
  appId: '<APP ID>',
  appKey: '<APP Key>'
 })
</script>

然后再在post.hbs里被注释掉的Comment部分添加这个就好了

<div class="col-md-12">
        <div id="vcomments"></div>
</div>

参考：Ghost 博客添加 Valine 评论系统

接入了 Google 广告#

为什么要接广告？

你当我们服务器是大风吹来的吗！

能补贴一下服务器的运营费用，顺便还能给文章作者一些报酬。

广告是 Google 根据你的兴趣个性化推荐的哦！所以每个人看到的都不太一样

下面是某位不愿意透露姓名的同学看到的情形

Google 广告接入还是很方便的

在 Google Adsense 申请就好，然后 Google 就会给一段神秘代码

像这样

<script async src="https://pagead2.googlesyndication.com/pagead/js/adsbygoogle.js"></script>
<script>
  (adsbygoogle = window.adsbygoogle || []).push({
    google_ad_client: "ca-pub-这里应该是你的个人编号",
    enable_page_level_ads: true
  });
</script>

然后就把这段代码扔进 Code Injection 的 Head 部分就好了

然后 Google 广告就会自动霸占你的网站了

记得在 Google Adsense 的设置里调整一下广告出现的频率，以免广告太多影响阅读体验。

参考：Google AdSense + Ghost

接入了统计系统#

我们站点一共接入了三个统计系统，分别是 Google Analytics，百度统计 和 不蒜子统计。

上面是百度统计的效果图，就是能分析一下站点的访问量什么的。

又没有多少人来统计个屁

百度和 Google 的接入都很方便，直接注册然后按提示获取代码就好了。

然后把获取到的代码扔进 Code Injection 的 Head 部分。

Head 部分都快成垃圾场了

不蒜子统计和前两个有点区别，这个是拿来展示的。

比如页面底部的访问量统计，每篇文章的“热度”都是用它做的。

方法也还算简单

先把这个填入 Code Injection 的 Head 部分

 <script async src="//busuanzi.ibruce.info/busuanzi/2.3/busuanzi.pure.mini.js"></script>

把这段代码扔进footer.hbs合适的地方就可以了

<span id="busuanzi_container_site_pv">本站总访问量<span id="busuanzi_value_site_pv"></span>次</span>

文章阅读量“热度”的方法也类似，改post.hbs就好了。
